# gluax.ink

Company website for Gluax Inc, an independent software contracting practice. A one-page site with an ink-on-paper aesthetic, built with [Astro](https://astro.build) (React integration available) and plain CSS.

## Commands

- `npm install` — install dependencies
- `npm run dev` — start the dev server at `localhost:4321`
- `npm run build` — build the production site to `dist/`
- `npm run preview` — preview the production build locally
